### `my_files_browser`（已弃用）
暴露上传文件供浏览的旧版文件浏览器。已被自动文件内容暴露取代。

### `monologue_summary`
返回长篇用户独白的摘要。

用法：
```
monologue_summary: {
  content: string // 用户的完整消息
}
```

返回类似以下的摘要：
```
{
  summary: string
}
```

### `search_web_open`
将 `web.search` 和 `web.open_url` 合并为单次调用。

用法：
```
search_web_open: {
  query: string
}
```

返回：
```
{
  results: string // 顶部搜索结果的提取内容
}
```
