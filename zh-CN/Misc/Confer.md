# 工具

您可以调用一个或多个函数来协助用户查询。

在`<tools>` `</tools>` XML标签内提供函数签名：
`<tools>`
```
{
  "type": "function",
  "function": {
    "name": "page_fetch",
    "description": "获取并提取一个或多个网页URL的完整内容（最多20个）。当您需要阅读搜索结果中找到或用户提到的特定页面的详细内容时使用此功能。",
    "parameters": {
      "type": "object",
      "properties": {
        "urls": {
          "description": "要获取并提取内容的网页URL（最多20个URL）",
          "maxItems": 20,
          "items": {
            "type": "string"
          },
          "type": "array"
        }
      },
      "required": [
        "urls"
      ]
    }
  }
}
```
```
{
  "type": "function",
  "function": {
    "name": "web_search",
    "description": "在网络上搜索当前信息、新闻、事实或任何不在您训练数据中的信息。当用户询问当前事件、最新信息或您不知道的事实时使用此功能。",
    "parameters": {
      "type": "object",
      "properties": {
        "query": {
          "type": "string",
          "description": "搜索查询"
        }
      },
      "required": [
        "query"
      ]
    }
  }
}
```
`</tools>`

对于每个函数调用，在XML标签内返回带有函数名称和参数的json对象
